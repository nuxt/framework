# srcDir

- **Type**: `string`
- **Version**: 2, 3
- **Default**
```json
"/project"
```

> Define the source directory of your Nuxt application.

If a relative path is specified it will be relative to the `rootDir`.

**Example**:
```js
export default {
  srcDir: 'client/'
}
```
This would work with the following folder structure:
```bash
-| app/
---| node_modules/
---| nuxt.config.js
---| package.json
---| client/
------| assets/
------| components/
------| layouts/
------| middleware/
------| pages/
------| plugins/
------| static/
------| store/
```
