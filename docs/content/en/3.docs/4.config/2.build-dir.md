# buildDir

- **Type**: `string`
- **Version**: 2, 3
- **Default**
```json
"/project/.nuxt"
```

> Define the directory where your built Nuxt files will be placed.

Many tools assume that `.nuxt` is a hidden directory (because it starts with a `.`). If that is a problem, you can use this option to prevent that.

**Example**:
```js
export default {
  buildDir: 'nuxt-build'
}
```
