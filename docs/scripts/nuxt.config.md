# Nuxt configuration file

Nuxt can be easily configured with a single `nuxt.config` file, which can have either a `.js`, `.ts` or `.mjs` extension.

```ts
import { defineNuxtConfig } from 'nuxt3'

export default defineNuxtConfig({
  // My Nuxt config
})
```
